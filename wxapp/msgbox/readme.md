- 组件思维
   弹窗组合了一些标签，形成了独立的弹窗功能，在其他页面里也要用到，
   组合成一个独立的组件，这个组件的名字就叫<dialog />
   写一个小程序，先分析主体，有那些组件，最后页面拼装

   页面是由组件拼装而成。
